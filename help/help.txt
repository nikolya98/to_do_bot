Описание бота:

«Бот Assistentem поможет Вам не забыть о важных событиях и делах!
Assistentem - это всего лишь очередная простая реализация «TO DO» списка...»


Команды:

/help - показать справку

/new_case <Название_задачи>: <Описание_задачи/комментарий> - добавить задачу
ВАЖНО: Символ даоеточия ":" после названия задачи обязателен

/show_all - показать все задачи

/show_case <Название_задачи> - показать задачу по названию
ВАЖНО: Без каких-либо знаков препинания (чтобы узнать название задачи используйте /show_all)

/show_this <Дата_создания> - показать задачу по дате названия
ВАЖНО: Дата указывается в формате дд-мм-гггг чч:мм:сс Пример: 27-09-2020 14:55:59
(можете воспользоваться командой /show_all, чтобы узнать дату созданию задачи)

/delete_case <Название_задачи> - удалить задачу по названию
ВАЖНО: Без каких-либо знаков препинания (чтобы узнать название задачи используйте /show_all)

/delete_this <Дата_создания> - удалить задачу по доте создания
ВАЖНО: Дата указывается в формате дд-мм-гггг чч:мм:сс Пример: 27-09-2020 14:55:59
(можете воспользоваться командой /show_all, чтобы узнать дату созданию задачи)
